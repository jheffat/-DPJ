## [1.1.0] - 2025-04-15
### Added
- Hashing support: single file, multiple files, and plain text hashing.
- Support for multiple hash algorithms (SHA-256, SHA-512, etc.).

### Changed
- Modularized hashing functionality in the DPJ CLI.

- File integrity check feature.
